Question: If you were to write that project as a modular program, what modules would you create?

Answer: Although I did not complete the Chapter 7 project noted here, it looks like there would be two key modules which should be included if the project were to be divided up as such:

  • Robot
  • Village

Question: Which module would depend on which other module, and what would their interfaces look like?

The Village module would depend on the Robot module, as the robot is a component of the village. The interface for the robot would have a runRobot, randomRobot, routeRobot, and goalOrientedRobot interface.

Question: Which pieces are likely to be available prewritten on NPM?

Answer: Given the size of NPM, there may already be a Robot package that does what we need.

Question: Would you prefer to use an NPM package or write them yourself?

Answer: I prefer to be in control and know what is going on, so I would most likely write it myself.
